# 번역의 민족 (BUNMIN) — 데모 페이지

백엔드·AI 모델 없이 브라우저만으로 서비스 전체 흐름을 시연하는 정적 데모입니다.
`index.html`을 **더블클릭**(file://)하거나 정적 서버로 열면 바로 동작합니다.

## 실행

```bash
# 방법 1: 그냥 더블클릭
demo/index.html

# 방법 2: 정적 서버 (레포 루트에서)
npx serve .   # → http://localhost:3000/demo/
```

## 구성

- **시작** — 실제 Start 페이지 재현 (이름 입력 → 수강자 입장 / 강의자 진입, 한/EN 전환)
- **강의자** — 슬라이드 업로드·번역 시뮬레이션, 오디오 테스트(레벨 미터), 강의 시작/일시정지/종료,
  판서·커서 스팟라이트, 원본/번역 토글, 용어집, 채팅, 자막 TXT/SRT 다운로드
- **수강자** — 대기 → LIVE → 일시정지 → 종료 상태, EN/KO 이중 자막 오버레이(스타일·크기·언어 설정),
  번역 슬라이드 뷰, 자료 다운로드, 채팅, 회선 품질 칩

하단 데모 스트립의 **▶ 데모 시작**을 누르면 약 70초짜리 강의 시뮬레이션이 자동 재생됩니다
(업로드 → OCR/번역 → 마이크 → ASR partial → 자막 → NMT → TTS → 판서/커서 동기화 → 일시정지 → 종료).
배속(0.5/1/2x), 리셋, 영어 TTS 소리(브라우저 음성 합성), 수강자 PiP 미리보기를 지원합니다.
자동 시연 중 화면을 직접 클릭하거나 입력하면 **시연이 자동으로 일시정지**되어 그대로 직접 조작할 수 있고,
▶ 를 다시 누르면 멈춘 지점부터 이어서 재생됩니다.

## URL 해시 옵션 (`+`로 조합)

| 해시 | 동작 |
|---|---|
| `#lecturer` / `#student` | 해당 뷰로 바로 시작 |
| `#dark` / `#gradient` | 테마 지정 |
| `#autoplay` | 강의자 뷰에서 시나리오 자동 재생 |
| `#autoplay-student` | 수강자 뷰에서 시나리오 관람 |

예: `index.html#autoplay-student+dark`

## 참고

- 프레임워크·빌드 없음. `js/*.js`가 전역 `window.BUNMIN` 네임스페이스를 공유합니다
  (`store.js`가 실제 앱의 `lectureStore.ts` 필드를 1:1 미러링).
- 슬라이드는 `js/slides.js`가 생성하는 한/영 SVG 목업입니다 (BST 강의 5페이지).
- 브랜드 폰트(paybooc/ELAND/학교안심)는 `assets/fonts/`에 동봉되어 있어
  demo/ 폴더만 단독으로 떼어가도 동일하게 렌더링됩니다.
  (paybooc·ELAND는 기업 배포 폰트이므로 외부 공개 배포 시 라이선스를 확인하세요.)
