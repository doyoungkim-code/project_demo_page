# 프로젝트 데모 페이지 모음

백엔드·빌드 없이 브라우저만으로 동작하는 두 프로젝트의 정적 데모와, 이 둘을 묶는 루트 랜딩 페이지입니다.
루트 [`index.html`](index.html)을 **더블클릭**(file://)하면 바로 열립니다. 외부 CDN·서버 없음.

## 구성

| 폴더 | 프로젝트 | 설명 |
|---|---|---|
| [`bunmin_demo/`](bunmin_demo/) | **번역의 민족 (BUNMIN)** | AI 기반 실시간 강의 번역 시스템 웹 데모 (시작 / 강의자 / 수강자 뷰) |
| [`ssabree_demo/`](ssabree_demo/) | **싸브리타임 (SSABREE)** | SSAFY 교육생 커뮤니티 Android 앱을 갤럭시 폰 프레임으로 재현한 데모 |

각 폴더의 `README.md`에 세부 기능·URL 해시 옵션이 정리되어 있습니다.

## 실행

```bash
# 방법 1: 그냥 더블클릭
index.html

# 방법 2: 정적 서버 (레포 루트에서)
npx serve .
```

## 공통 사항

- 프레임워크·빌드 없음. 순수 HTML/CSS/JS.
- 각 데모 하단의 데모 스트립에서 **▶ 데모 시작**을 누르면 자동 시연이 재생됩니다.
  시연 중 화면을 직접 클릭/입력하면 자동으로 일시정지되어 직접 조작할 수 있고, ▶ 로 이어서 재생됩니다.
