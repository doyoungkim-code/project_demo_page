# 싸브리타임 (SSABREE) — 데모 페이지

백엔드 없이 브라우저만으로 싸브리타임의 전체 기능 흐름을 시연하는 정적 데모입니다.
Android 앱(Jetpack Compose)의 UI를 갤럭시 폰 프레임 안에 재현했습니다.
`index.html`을 **더블클릭**(file://)하면 바로 동작합니다. 외부 리소스(폰트·CDN·이미지) 없음.

## 실행

```bash
# 방법 1: 그냥 더블클릭
ssabree_demo/index.html

# 방법 2: 정적 서버 (레포 루트에서)
npx serve .   # → http://localhost:3000/ssabree_demo/
```

## 구성

- **폰 1 (김도영, 14기 구미 캠퍼스)** — 주인공 시점. 스플래시 → 로그인 → 홈 대시보드 →
  게시판(Hot·투표·댓글/대댓글·AI 검열) → 그룹 매칭(지원) → 쪽지(실시간 채팅) → 알림 → 마이페이지/포트폴리오
- **폰 2 (박싸피, 스터디장)** — 채팅·그룹 지원 수락 등 **양방향 장면에서만 슬라이드 인**
- 두 폰이 하나의 스토어를 구독하므로 채팅/알림/수락이 실시간으로 동기화됩니다 (WebSocket·FCM 흐름 재현)

하단 데모 스트립의 **▶ 데모 시작**을 누르면 약 110초짜리 전체 기능 시연이 자동 재생됩니다
(로그인 → 홈 → 투표/댓글 → 싸피봇 AI 검열 → 스터디 지원 → 리더 수락(FCM 헤드업) → 실시간 쪽지 → 알림/포트폴리오).
배속(1/2/0.5x), 리셋, 라이트/다크 토글, 폰 2대 수동 표시를 지원하며,
**일시정지 중에는 폰을 직접 클릭하며 자유롭게 탐색**할 수 있습니다.
자동 시연 중 폰을 직접 클릭하거나 입력하면 시연이 **자동으로 일시정지**되고,
▶ 를 다시 누르면 멈춘 지점부터 이어서 재생됩니다.

## URL 해시 옵션 (`+`로 조합)

| 해시 | 동작 |
|---|---|
| `#autoplay` | 로드 즉시 시나리오 자동 재생 |
| `#dark` | 다크 테마로 시작 |
| `#dual` | 폰 2대(박싸피 포함)로 시작 |

예: `index.html#autoplay+dark`

## 참고

- 프레임워크·빌드 없음. `js/*.js`가 전역 `window.SSABREE` 네임스페이스를 공유합니다
  (bunmin_demo와 동일한 구조: store.js pub/sub + scenario.js 가상시간 스케줄러).
- 색상 토큰은 `android/.../core/designsystem/theme/Color.kt`를 그대로 복제했습니다 (`css/tokens.css`).
- 로고·아이콘·solved.ac 티어 배지는 전부 인라인 SVG로 재현되어 폴더 단독 배포가 가능합니다.
- 게시판 작성자 표기는 실제 앱과 동일하게 익명(싸용자 / 싸용자N / 싸용자(작성자))입니다.
- AI 검열 데모: 글에 부적절한 표현이 포함되면 등록 2.6초 후 싸피봇이 블라인드 처리합니다
  (실제 서비스의 서버측 Gemini 비동기 검열 흐름 재현).
